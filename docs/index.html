A Complete Audio Ecosystem

The Core

Batch Studio (NLE)

Don't just generate lines; build scenes. The block-based editor allows you to mix multiple speakers, clone voices, and custom designs into a single script.

Features a "Traffic Light" system to track generation status and a smart stitching engine to render full conversations instantly.

Automation

Dynamic Plugin System

The first extensible AI audio suite. Drop new features into the modules/ folder to add custom tabs, automation scripts, and entirely new workflows without touching the core code.

Supports Headless Integration: Build background services like Watch Folders or Local HTTP APIs to generate audio from external applications.

The Replica

Professional Voice Cloning

Turn any audio file into a digital actor. The integrated cloning engine uses "Lock Voice" technology to cache speaker profiles, allowing for instant generation without re-processing.

Includes a Multi-Part Mode to handle long scripts by splitting them automatically.

Workflow

Transcript Helper

Good data equals good audio. The Prep Station includes a built-in Whisper AI engine to auto-transcribe your reference files.

Record from your microphone or system audio, crop the waveform visually, and send it directly to the cloner.
